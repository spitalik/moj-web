# Dungeon Miner — Neon Cyber

Simple browser game (HTML/CSS/JS). Move with arrow keys, press Space to mine adjacent tiles, collect ore for score. Restart with `R`.

How to run:
1. Open `dungeon-miner/index.html` in a modern browser.
2. Allow audio context interaction by pressing a key or clicking if sounds do not play automatically.
